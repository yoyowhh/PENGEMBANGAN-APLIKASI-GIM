# PENGEMBANGAN-APLIKASI-GIM
Repository ini berisi proyek dan tugas dari mata kuliah **Pengembangan Aplikasi Gim**.  
Proyek dalam repository ini dikembangkan menggunakan **Unity** dengan bahasa pemrograman **C#** untuk membuat permainan 2D/3D sederhana.

## 📌 Deskripsi
Repository ini digunakan untuk menyimpan source code, aset, serta dokumentasi selama proses pembuatan gim pada mata kuliah Pengembangan Aplikasi Gim.

Proyek ini mencakup beberapa fitur dasar dalam pengembangan game seperti:
- Player movement
- Sistem skor
- Collision detection
- Spawn object
- UI sederhana

## 🛠️ Tools & Teknologi
- **Unity Engine**
- **C#**
- **Visual Studio Code / Visual Studio**
- **Git & GitHub**
